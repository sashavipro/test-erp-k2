Створити невеликий сервіс на Python.

Використати можна: fastapi
Потрібні сутності

    Клієнт
    Товар
    Замовлення

Мінімальний функціонал

Реалізувати можливість:

    створити клієнта;
    створити товар;
    створити замовлення;
    отримати список замовлень по клієнту;
    автоматично порахувати суму замовлення.

Обов’язкові бізнес-правила

    не можна створити замовлення без клієнта;
    у замовленні має бути хоча б один товар;
    сума замовлення має розраховуватися автоматично на основі товарів.

База даних

Можна використати:PostgreSQL
Що має бути в результаті

    код проєкту;
    інструкція запуску;
    короткий опис структури;
    коротке пояснення, чому ви обрали саме такий підхід.

Буде плюсом: Docker, SQLAlchemy, валідація даних, прості тести, акуратна структура проєкту, API у зрозумілому форматі, найпростіший фронтенд(htmx, TypeScript,css,html)
